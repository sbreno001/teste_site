Resumo rápido de quando criar uma nova conta ou um container na mesma conta:

Conta: Usada para organizar sites ou clientes diferentes. Cada negócio ou site independente deve ter uma conta separada.
Container: Usado para gerenciar as tags de um site ou aplicativo específico. Se for o mesmo site, geralmente um container serve, mas se houver partes distintas (como checkout), pode-se usar containers separados.
-----------------------------------------------------------------------------------------------------
Tags:
O que são: São trechos de código ou scripts que você deseja adicionar ao seu site ou aplicativo. As tags são responsáveis por realizar tarefas específicas, como rastrear eventos, enviar dados para ferramentas de análise (como o Google Analytics), realizar campanhas de remarketing, entre outras.
Exemplo: Uma tag pode ser um código de rastreamento do Google Analytics, um pixel do Facebook ou uma tag de conversão do Google Ads.
Acionadores (Triggers):
O que são: São as condições ou eventos que determinam quando e onde uma tag será executada. Ou seja, os acionadores controlam o comportamento das tags, dizendo a elas quando devem ser ativadas.
Exemplo: Você pode configurar um acionador para disparar uma tag de Google Analytics quando um visitante clicar em um botão específico ou quando uma página for carregada.
